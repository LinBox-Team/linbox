LinBox examples
Directory (linbox)/examples/


To start running LinBox codes:
 
1. Create a file
example.C
  as those given below.

2. Using the Makefile found in the corresponding subdirectory of /linbox/examples/ compile your code:

make example

3. Run

example

Example files:
- ex_integers.C     [(linbox)/examples/beginners/integers]

- Finite fields   [(linbox)/examples/beginners/fields]
