How to install LinBox
(www.linalg.org)



 

1. Go to the newly-created linbox directory and run the script autogen.sh, optionally specifying the installation directory or an optional package directory using prefixes:
> cd linbox
> ./autogen.sh [--prefix=<prefix> --with-package-prefix=<package>]
This sets up the library and runs  ./configure

Some possible prefixes:

- Where to put the include/ and lib/ directories:
--prefix <directory>
Ex:  --prefix `pwd` will put the files within the linbox directory
- The use of an optional package is switched on by specifying the directory where it can be found:
--with-package-prefix=<package>
Ex:   --with-gmp-prefix=<gmp_directory>    (Gmp)
    --with-givaro-prefix=<givaro_directory>   (Givaro)
    --with-saclib-prefix=<saclib_directory>  (Saclib)
    --with-ntl-prefix=<ntl_directory>    (NTL)
    --with-alp-prefix=<alp_directory>    (ALP)
         --with-lidia-prefix=<lidia_directory>  (Lidia)


2.  Install the library:

>  make install
