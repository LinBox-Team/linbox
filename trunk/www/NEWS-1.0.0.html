Announcing LinBox version 1.0.0

July 8, 2005

LinBox 1.0.0 is our first "stable" release. The solutions section has undergone a major revision and extension with the purpose of making the linear algebra functions easily invoked. Also the examples directory contains illustrative uses of each function. This release is prepared for availability at ISSAC '2005.
It has been checked with g++-3.4, g++-4.0 on linux PIV, WinNT cygwin PIV, ... with givaro-3.1.3, gmp-4.1.4, ntl-5.4, ATLAS-3.7.10.

What's new in this release:

  • Dixon's Rational solving.
  • Characteristic polynomial.
  • Positive (semi)definiteness test.
  • A block Lanczos solver.
  • Hybrid Smith form algorithm, including a program for invoking smith form algorithms in the examples directory.
  • the "linbox-program" in examples, providing easy access to some algorithms applied to matrices read from files.
  • Other new algorithms especially elimination methods and block algorithms in various states of usability.
  • More integer matrix capabilities, with simple drivers in the solutions directory.
  • ...and miscellaneous.
