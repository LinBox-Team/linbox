Other projects of interest

  • The GNU MultiPrecision Bignum Library is required by LinBox.
  • BLAS: basic linear algebra subprograms are required by LinBox.
    • Goto BLAS: very fast assembly optimized BLAS
    • ATLAS: Automatically Tuned Linear Algebra Software provides very good portable BLAS.
    Note that LAPACK is used in parts of LinBox.
  • NTL: A Library for doing Number Theory is used in parts of LinBox
  • Givaro is used in many parts of LinBox
  • Fflas-Ffpack is used for dense linear algebra
  • IML used for fast integer/rational linear algebra.
  • gnuplot used for creating plots in benchmarks.
