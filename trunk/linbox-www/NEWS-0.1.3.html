Announcing LinBox version 0.1.3

27 August 2002

LinBox 0.1.3 is a development release, adding several new algorithm implementations and refactoring some existing interfaces.

What's new in this release:

  • Determinant of an integer matrix, using Chinese remaindering
  • Refactoring of the Wiedemann-based system solver, with support for inconsistency certificate, preconditioning, and random solution to singular systems
  • Implementation of the Lanczos iteration, with support in solve
  • Refactoring of Butterfly
  • Refactoring of VectorFactory into VectorStream, with an istream-style interface
  • Imported code for the CEKSTV switch for the butterfly preconditioner, and wrote a test suite
  • New Matlab-style sparse matrix output format
  • Fixes to sparse matrix RawIterator
  • RawIndexIterator support in DenseMatrix
  • RawIterator and RawIndexIterator support in NagSparse
  • Have the tests for fields other than Modular run the complete test suite
  • New example in examples/blackbox that loads a blackbox and computes its minimal polynomial
  • Fixes to the commentator's reporting features

Known bugs

  • The butterfly preconditioner does not work when invoked inside solve, though its standalone test suite works
